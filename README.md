Smart Video Intercom System with Vibration Alert

Overview
This project is a Smart Video Intercom System designed to improve communication accessibility, especially for deaf and hard-of-hearing individuals.

The system combines:
Visual communication (video interface)
Vibration-based alert mechanism

to enable seamless interaction within indoor environments like offices, homes, or institutions.

Problem Statement

Traditional intercom systems rely heavily on audio communication, which makes them unsuitable for:

Deaf individuals
People with hearing impairments
Situations where audio communication is not feasible

This project aims to provide an inclusive communication solution using visual and physical alert mechanisms.

Proposed Solution

The system consists of:

A video-based communication interface
A vibration sensor module to detect activity (e.g., knocking, emergency signals)
A trigger mechanism to notify the receiver visually

When a vibration is detected:

A signal is sent to the system
An alert is triggered on the receiver’s interface
Video communication is initiated for interaction

Features
Real-time visual alert system
Dual interface for two-way communication (Room A & Room B)
Camera-based video preview
Scalable for IoT-based integration
